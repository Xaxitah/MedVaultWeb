Marcha · Biotipo · Atitude · Fácies

1. Visão Geral

O exame físico geral (exploración física general) registra os dados observáveis que refletem o estado geral do paciente antes do exame regional e por sistemas.

Os 12 componentes do EF geral (Professor Trans 6):

| # | Componente | Parte do Tema |

|---|-----------|--------------|

| 1 | Marcha | Parte 1 |

| 2 | Biotipo | Parte 1 |

| 3 | Atitude/postura | Parte 1 |

| 4 | Fácies | Parte 1 |

| 5 | Talha | Parte 2 |

| 6 | Peso / IMC | Parte 2 |

| 7 | Temperatura | Parte 2 / Tema-05-Sinais-Vitais |

| 8 | Pele | Parte 2–3 |

| 9 | Mucosas | Parte 3 |

| 10 | Pâneras | Parte 3 |

| 11 | TCS / edema | Parte 3 |

| 12 | Panículo adiposo | Parte 3 |

Condições para o EF: iluminação adequada (natural preferida); temperatura ambiente; privacidade; consentimento do paciente; expor apenas a zona examinada. Médico sempre à direita do paciente.

2. Núcleo Obrigatório

  • Marcha normal: "deambula sem dificuldade" — registro padrão

  • Biotipo: normolíneo / brevilíneo / longilíneo — 3 tipos básicos; ângulo costofrênico diferencia

  • Atitude em pé: classificação de Lloyd T. Brown (A=excelente, D=má)

  • Decúbito: ativo indiferente / ativo forçado / passivo

  • Fácies normal: fácies harmônica — expressão simétrica, sem alterações

  • Fácies patológicas comuns: adenoidea, hipocrática, renal, mixedematosa, basedowiana, parkinsoniana, paralisia facial

⚠️ ATENÇÃO: Ao examinar a fácies, comparar sempre o lado esquerdo vs direito do rosto — assimetrias são dados semiológicos importantes.

3. Explicação Detalhada

3.1 Marcha (Marcha)

Definição: forma de caminhar; observada desde a entrada do paciente no consultório.

Normal: "deambula sem dificuldade" — registro no prontuário.

Marchas patológicas

| Nome | Descrição | Causa típica |

|------|-----------|-------------|

| Parkinsoniana | Passos curtos, tronco inclinado para frente, braços sem balançar, "marcha en festinación" | Doença de Parkinson |

| Espástica / guadañante | Membro inferior estendido, move em arco de fora para dentro (movimento de foice/guadaña) | Hemiplejia espástica (AVC) |

| Cerebelosa / titubeante | Marcha em zigue-zague, ebriosa, com desvio lateral (lateropulsão) | Síndrome cerebelosa |

| Digitálica | Caminha nas pontas dos pés | Problema de quadril |

| Pato (anserina) | Oscilação lateral do tronco ("pato") | Problemas musculares ou de quadril |

| Escarvante / estepaje | Levanta muito a perna; pé "cai" (pé caído) | Problema neurológico (n. ciático poplíteo) |

| Atáxica / taloneante | Projeta o pé bruscamente à frente; assenta o calcanhar com força | Tabes dorsal (neurossífilis) |

| Polineurítica / estepaje | Eleva a perna amplamente, pé pendurado | Polineurites (diabética, alcoólica, arsenical) |

3.2 Biotipo (Biotipo / Hábito externo)

Definição: constituição corporal geral do indivíduo — herdada e moldada pelo ambiente.

3 tipos básicos (classificação italiana — Llanio)

| Biotipo | Sinonímias | Morfologia | Ângulo costal | Musculatura |

|---------|-----------|-----------|--------------|-------------|

| Normolíneo | Normotipo | ~1.70m / ~70kg; proporções equilibradas | Médio (~90°) | Bem desenvolvida |

| Brevilíneo | Pré-linfático / pícnico / hipervegetativo / macrosplácnico | Baixo, corpulento, pescoço curto; tronco > membros | Obtuso (>90°) | Bem desenvolvida |

| Longilíneo | Asténico / microesplácnico / hipovegetativo | Alto, magro, pescoço longo; membros > tronco; tórax plano/estreito | Agudo (<90°) | Pouco desenvolvida |

Tipos adquiridos (Professor Trans 6):

  • Hiperestênico: ganho muscular por musculação — não confundir com brevilíneo genético

  • Pícnico: obesidade genética central, <1.5m — coincide com brevilíneo mas de origem genética

Predisposições clínicas (Llanio Cap 24):

  • Longilíneo → predisposição a tuberculose pulmonar, hipertireoidismo, doenças nervosas

  • Brevilíneo → predisposição a DM, gota, HAS, arteriosclerose, AVC hemorrágico

3.3 Atitude / Postura (Actitud)

Posição em pé — Classificação de Lloyd T. Brown

| Grau | Postura | Descrição |

|------|---------|----------|

| A | Excelente | Coluna alinhada, tórax alto, abdome plano, cabeça ereta |

| B | Boa | Ligeiro abaixamento de um parâmetro |

| C | Má | Tórax caído, abdome projetado |

| D | Péssima | Colapso postural marcado — todos os parâmetros alterados |

Posição deitada — Decúbitos

| Tipo | Quem decide | Características |

|------|------------|----------------|

| Ativo indiferente | O próprio paciente (livre escolha) | Muda de posição à vontade; pouco interesse semiológico |

| Ativo forçado | Paciente (obrigado pela doença) | Posição alivia dor ou dispneia |

| Passivo | Gravidade (paciente não consegue mover-se) | Inconsciente, coma, paralisia extensa |

Decúbitos por nome:

  • Supino/dorsal: de costas (face para cima)

  • Prono/ventral: de bruços (face para baixo)

  • Lateral direito/esquerdo: apoiado no respectivo lado

Decúbitos ativos forçados (patológicos)

| Nome | Posição | Causa |

|------|---------|-------|

| Ortopneia | Sentado/semisentado, apoiado em almofadas | Insuficiência ventricular esquerda, asma grave |

| Signo do travesseiro | Sentado, apoiado num travesseiro nos joelhos | Derrame pericárdico volumoso |

| Plegaria mahometana | Ajoelhado, tronco inclinado à frente, apoiado nos cotovelos | Pericardite, tamponamento cardíaco |

| Decúbito lateral forzado | Lateral — lado são (pleurite inicial) / lado enfermo (derrame volumoso) | Pleurite, pneumotórax, cavernas pulmonares |

| Decúbito dorsal | Supino, com flexão dos membros inferiores | Abdome agudo (apendicite, peritonite) |

| Decúbito prono | Face para baixo | Cólica abdominal, úlcera posterior do estômago, mal de Pott |

Atitudes patológicas especiais (neurológicas)

| Nome | Descrição | Causa |

|------|-----------|-------|

| Opistótonos | Hiperextensão posterior — apoia em cabeça + calcanhares, dorso arqueado para cima | Tétano, estricnina, meningite, epilepsia |

| Emprostótonos | Curvatura anterior — apoia em fronte + pés | Tétano (raro — "tétano em bola") |

| Pleurotótonos | Curvatura lateral | Tétano, lesões de coluna/pleura |

| Gatillo de escopeta | Decúbito lateral, extensão de nuca, flexão+adução das coxas — silhueta de gatilho | Meningite tuberculosa |

Actitudes patológicas em pé (Cap 24)

  • Hemiplejia: ombro caído, MS em flexão e adução, MI em extensão

  • Parkinson: tronco levemente inclinado para frente, rigidez, fácies imóvel

  • Actitude de tenor: hiperlordose lombar, tronco estendido para trás — miopatias primitivas

3.4 Fácies (Fácies)

Definição: expressão global do rosto; reflete estado emocional e doenças.

Normal: fácies harmônica — expressão facial simétrica, sem alterações evidentes.

Técnica: comparar sempre lado esquerdo vs. lado direito.

Fácies patológicas — por sistema

Respiratório:

| Fácies | Características |

|-------|----------------|

| Adenoidea | Boca permanentemente aberta, mandíbula inferior caída, nariz fino com narinas pouco desenvolvidas, expressão de pouca inteligência | Vegetações adenóideas, obstrução nasofaríngea |

| Neumônica | Mirada brilhante/ansiosa, rubor em uma bochecha, aleteo nasal, herpes labial | Pneumonia |

Cardiovascular:

| Fácies | Características |

|-------|----------------|

| Aórtica | Palidez, latidos visíveis nas artérias temporais e cervicais, sinal de Musset (cabeça balança em "sim") | Insuficiência aórtica |

| Ansiosa da asistolia | Cianose pronunciada, edema facial, expressão de ansiedade, aleteo nasal | Insuficiência cardíaca descompensada |

| Mitral | Bochechas violáceas/cianóticas — "manchas mitrais" | Estenose mitral |

Renal:

| Fácies | Características |

|-------|----------------|

| Renal / edematosa | Edema dos pálpebros (estreita a fissura palpebral), palidez, face arredondada — "fácies de boneco chinês" | Nefrite, síndrome nefrótica |

Digestivo/Geral grave:

| Fácies | Características |

|-------|----------------|

| Peritoneal / hipocrática | Expressão ansiosa, palidez terrosa, olhos e bochechas escavados, "orlados de olheiras violáceas", nariz afilado, saliente dos pômulos | Peritonite, abdome agudo grave |

| Hepática | Tinte amarelo/palha (icterícia), arañas vasculares (telangiectasias aracnoideas) | Doença hepática crônica |

Endócrino:

| Fácies | Características | Causa |

|-------|----------------|-------|

| Acromegálica | Mandíbula inferior prognata, nariz alargado, pômulos salientes, diastema dentário, mãos/pés grandes | Hipersecreção GH (hipófise) |

| Cretinóidea | Face larga, lábios grossos, língua protrusa, nariz achatado, pele seca/pardacenta, expressão estúpida | Hipotireoidismo fetal / cretinismo |

| Mixedematosa | Pele rugosa, seca, espessa; pálpebros abotagadas; lábios grossos; ausência da cauda das sobrancelhas (signo da cauda) | Hipotireoidismo adulto (mixedema) |

| Basedowiana / hipertiróidea | Expressão de terror/susto, exoftalmo (olhos saltantes), abertura palpebral aumentada; sinais: Stellwag (↓ piscar), Dalrymple (abertura palpebral), Von Graefe (pálpebro superior atrasa ao olhar para baixo) | Hipertireoidismo (Basedow) |

Neurológico:

| Fácies | Características |

|-------|----------------|

| Paralisia facial | Desvio da boca para o lado são, ptose palpebral no lado paralisado; Sinal de Bell (olho não fecha — paralisia periférica) | Paralisia periférica do VII par |

| Parkinsoniana | Imóvel, inexpressiva, máscara — olhos inteligentes em contraste | Doença de Parkinson |

| Mediastinal | Cianose + edema violáceo na face, pescoço, membros superiores e tórax superior — "cianose em esclavina" | Compressão da veia cava superior |

Infecciosas/gerais:

| Fácies | Características |

|-------|----------------|

| Febril | Mirada brilhante, rubor malar, palidez do restante da face | Febre em geral |

| Tetânica | Riso sardônico (contração muscular fixa), trismo (impossibilidade de abrir a boca) | Tétano |

| Tifóidica / estuporosa | Expressão de sonolência/sopor, olhos semicerrados, lábios secos com fuliginosidades | Febre tifoide, toxiinfecções graves |

Emocionais (Trans 6):

  • Tensa/ansiosa, interrogadora, alegre, colérica, triste — dão pistas sobre o estado psicoemocional do paciente

5. Comparação entre Conceitos Parecidos

Decúbito Ativo Forçado vs Passivo

| Critério | Ativo Forçado | Passivo |

|---------|--------------|---------|

| Quem decide a posição | O paciente (obrigado pela doença) | Gravidade (paciente não consegue se mover) |

| Consciência | Preservada | Geralmente alterada |

| Causa típica | Ortopneia (IC), dor intensa | Coma, hemorragia cerebral, choque grave |

| Valor semiológico | Alto (indica o sistema comprometido) | Alto (indica gravidade extrema) |

Biotipo Longilíneo vs Brevilíneo

| Critério | Longilíneo | Brevilíneo |

|---------|-----------|-----------|

| Estatura | Alto | Baixo |

| Tórax | Estreito, aplanado, ângulo agudo | Amplo, ângulo obtuso |

| Musculatura | Pouco desenvolvida | Bem desenvolvida |

| Órgãos | Pequenos, ptose visceral | Desenvolvidos, bem posicionados |

| Predisposição | TB, hipertireoidismo, neuroses | DM, gota, HAS, arteriosclerose |

Fácies Mixedematosa vs Basedowiana (opostas)

| Critério | Mixedematosa | Basedowiana |

|---------|-------------|------------|

| Tireóide | Hipofunção | Hiperfunção |

| Olhos | Pálpebros abotagadas | Exoftalmo, abertura aumentada |

| Pele | Seca, rugosa, espessa | Normal/suada |

| Expressão | Apática, estúpida | Terror, susto |

| Sobrancelhas | Sem cauda (signo da cauda) | Normais ou arcadas |

9. Estratégia de Memorização

Para biotipos:

"N-B-L" = Normal-Brevilíneo-Longilíneo

Ângulo: "Normal = Noventa; Brevilíneo = Bem aberto (obtuso); Longilíneo = Lima aguda (agudo)"

Para decúbitos:

I-F-P = Indiferente (livre) / Forçado (doente escolhe) / Passivo (gravidade decide)

Para fácies basedowiana — 3 sinais:

S-D-V = Stellwag (↓ piscar) / Dalrymple (abertura ↑) / Von Graefe (pálpebro atrasa)

Para ortopneia:

"IC esquerda → pulmão cheio de sangue → senta para drenar pela gravidade"

Para fácies hipocrática:

"PALE ORLAS" = Palidez terrosa / Ansiedade / Lábios / Escavados (olhos) / Ojeras = "o rosto da morte iminente"

Para marchas — mnemônico:

"P-E-C-D-P-E-A-P" = Parkinson / Espástica / Cerebelosa / Digitálica / Pato / Escarvante / Atáxica / Polineurítica
